# Basic Calculator (Python + Tkinter)

A simple desktop calculator built using Python and Tkinter. This project demonstrates how to create a graphical user interface (GUI) and perform basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, and division
- Simple and user-friendly interface
- Button-based input
- Basic error handling (e.g., division by zero)

## Tech Stack

- Python
- Tkinter

## Project Structure

calculator/
│── main.py
│── README.md

## Requirements

- Python 3.x
- Tkinter (usually included with Python)
